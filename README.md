# Mini MES 프로젝트 (React + Spring Boot + MyBatis + PostgreSQL)

React, Spring Boot, MyBatis, PostgreSQL을 활용한 제조 실행 시스템(MES) 미니 프로젝트.

## 기술 스택

- 프론트엔드: React
- 백엔드: Spring Boot (MyBatis)
- 데이터베이스: PostgreSQL
- 빌드 도구: Maven
- 테스트: JUnit 5 (API 통합 테스트)

## 주요 기능

- 제품, 자재, 공정, 설비 등 MES 기준정보 관리
- 생산지시 및 작업지시 관리
- 생산실적, 불량실적 등 실적 관리
- 공정 및 설비 이력 관리
- 사용자 및 권한 관리
- 기본 CRUD API 제공
- React 기반의 웹 UI 제공
